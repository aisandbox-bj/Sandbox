Sandbox — Encrypt Dashboard Local tool

Wraps an HTML dashboard in AES-GCM encryption so it can sit safely on a public GitHub Pages site.

How this works. Pick one or more .html files, set a passphrase (12+ chars), and click Encrypt. Each file becomes a .enc.json envelope you can drop into /private/ on the repo. The tool runs entirely in your browser — nothing is uploaded.
